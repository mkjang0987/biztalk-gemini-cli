# [PRD] 업무 말투 변환기 (Business Tone Converter)

**버전**: 1.0  
**상태**: 초안 (Draft)  
**작성일**: 2026-04-22  
**프로젝트 성격**: 입문 개발자를 위한 One Day 실습 프로젝트

---

## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 사용자가 입력한 일상적인 문구를 AI(Upstage Solar-Pro2)를 활용하여 비즈니스 상황에 적합한 격식 있는 말투로 변환해주는 웹 서비스입니다. 사회 초년생이나 비즈니스 커뮤니케이션에 어려움을 느끼는 직장인들이 시간을 단축하고 전문성을 높일 수 있도록 돕습니다.

## 2. 제품 목표 (Goals)
- **핵심 가치**: 고민하는 시간을 줄이고, 상황에 맞는 정확한 비즈니스 언어 제공.
- **실습 목표**: API 연동, 백엔드(FastAPI)와 프론트엔드(HTML/JS)의 기본 흐름 이해 및 배포 경험.

## 3. 사용자 및 사용자 스토리 (User Personas & Stories)
- **사용자**: 신입 사원, 고객 응대 담당자, 협업이 잦은 실무자.
- **사용자 스토리**: "나는 상사에게 보고서 지연을 정중하게 알리고 싶다. 하지만 실례가 될까 봐 문장을 고치는 데 30분째 고민 중이다. 핵심 내용만 입력하면 AI가 격식 있는 문장을 만들어 주었으면 좋겠다."

## 4. 핵심 기능 요구사항 (Functional Requirements)

### 4.1. 메인 인터페이스 (UI)
- [ ] **텍스트 입력창**: 변환하고 싶은 내용을 자유롭게 입력할 수 있는 공간 (Textarea).
- [ ] **수신 대상 선택 (라디오 버튼 또는 버튼형)**: 👔 상사/임원, 🤝 타팀 동료, 🌐 고객/외부, 👥 팀 내 동료 중 택 1.
- [ ] **변환 버튼**: 입력값과 선택된 대상을 서버로 전송.
- [ ] **결과 출력창**: AI가 생성한 문장을 보여주는 공간.
- [ ] **복사하기 버튼**: 생성된 문장을 클릭 한 번으로 클립보드에 복사.

### 4.2. 인공지능 변환 로직 (AI Logic)
- [ ] **프롬프트 엔지니어링**: 선택된 수신 대상에 따라 AI가 다른 어조(Tone & Manner)를 생성하도록 시스템 프롬프트 설정.
    - **상사**: 공손함, 격식, 보고 중심.
    - **타팀**: 정중함, 협조 요청, 명확성.
    - **고객**: 신뢰감, 친절함, 사과/안내 문구.
    - **동료**: 간결함, 실무적, 명확한 액션 아이템.

### 4.3. 백엔드 API (FastAPI)
- [ ] `POST /convert`: 프론트엔드로부터 `text`와 `target` 데이터를 받아 Upstage API 호출 후 결과 반환.

## 5. 비기능 요구사항 (Non-functional Requirements)
- **속도**: 변환 요청 후 3~5초 이내에 결과가 화면에 표시되어야 함.
- **사용성**: 별도의 설명 없이도 즉시 사용할 수 있는 직관적인 단일 페이지 구성.
- **배포**: GitHub을 통한 버전 관리 및 Vercel(또는 유사 플랫폼)을 통한 공용 URL 제공.

## 6. 기술 스택 (Tech Stack)
- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (Fetch API)
- **Backend**: Python 3.10+, FastAPI
- **AI Model**: Upstage Solar-Pro2 (via LangChain 연동 권장)
- **Deployment**: Vercel (Frontend/Backend) or Render

## 7. 상세 사용자 흐름 (User Flow)
1. 사용자가 웹 페이지 접속.
2. 수신 대상(예: 고객/외부)을 클릭하여 선택.
3. 입력창에 "물건 배송이 좀 늦어질 것 같아요 미안해요" 입력.
4. [변환하기] 버튼 클릭.
5. 로딩 애니메이션 표시 (선택 사항).
6. 결과창에 "안녕하세요. 주문하신 상품의 배송이 당초 예정보다 다소 지연될 것으로 파악되어 안내드립니다..." 문구 노출.
7. [복사하기] 버튼을 눌러 메시지 전송에 활용.

## 8. 제외 범위 (Out of Scope)
- 회원가입 및 로그인 기능.
- 변환 기록 저장 및 관리 기능 (DB 사용 안 함).
- 다국어 변환 (한국어 전용).
- 복잡한 애니메이션이나 고도의 반응형 디자인.
